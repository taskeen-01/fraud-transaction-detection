# Fraud Transaction Detection System

## Project Overview

This project focuses on detecting fraudulent transactions using machine learning techniques. 
The goal is to identify suspicious activities and provide actionable business insights to reduce financial loss.


## Dataset Understanding

* Loaded transaction dataset using Python
* Explored dataset structure and features
* Checked for missing values and data quality


## Exploratory Data Analysis (EDA)

* Analyzed fraud vs non-fraud distribution
* Studied transaction amount patterns
* Identified fraud trends based on time and location
* Visualized insights using Matplotlib and Seaborn


## Feature Engineering

Created new features to improve model performance:

* Transaction hour extraction
* Spending behavior patterns
* Detection of abnormal transactions


## Machine Learning Models

Implemented and compared multiple models:

* Logistic Regression
* Decision Tree
* Random Forest

### Evaluation Metrics:

* Accuracy
* Precision
* Recall
* Confusion Matrix


## Key Findings

* Fraud transactions are highly imbalanced compared to normal transactions
* Certain time periods show higher fraud activity
* High-value transactions have higher fraud probability


## SQL Business Analysis

SQL queries were written to answer business-driven questions such as:

* Fraud rate by location
* High-risk transaction patterns
* Time-based fraud trends
